# Industrial Dashboard UI Recommendations

The frontend will remain a Next.js React application, but the UI must be transformed into a premium, "wow-factor" Industrial Dashboard.

## 1. Design Aesthetics (The "Premium Feel")
- **Theme**: Dark mode by default (industrial control room vibe). Use deep slate grays (`#0f172a`), neon accents (cyan `#06b6d4`, warning amber `#f59e0b`, critical red `#ef4444`).
- **Typography**: `Inter` for data density, `Orbitron` or `Rajdhani` for headers/KPIs to give an industrial/technical feel.
- **Glassmorphism**: Subtle translucent panels with background blur for floating elements like the Copilot chat.
- **Micro-animations**: Hover states on graph nodes, pulsing indicators for live sensor data, smooth slide-ins for alerts.

## 2. Core Dashboard Components

### The Global Command Center
- High-level KPIs: Plant Health Score, Active Incidents, Pending Compliance Audits.
- A live activity feed showing agents working in the background (e.g., "Document Agent ingested Manual_P101.pdf", "Lessons Learned Agent identified new risk").

### Knowledge Graph Explorer (Using React Flow)
- An interactive visual map of the facility.
- Users can click on a `Plant` -> zooms to `Areas` -> zooms to `Equipment`.
- Clicking an `Equipment` node opens a side panel showing connected `Manuals`, `Incidents`, and `Sensors`.

### The Industrial Copilot (Chat Interface)
- A persistent floating action button or sidebar.
- **Rich Responses**: Not just text. When the Copilot answers, it should render `VisualizationBundle` elements (Recharts) natively in the chat (e.g., a line chart of failure probabilities).
- **Citations**: Inline reference chips that, when clicked, open the exact page of the PDF manual in a modal.

### Document Center
- A drag-and-drop ingestion zone for PDFs, CADs, and CSVs.
- Progress bars showing OCR, Chunking, and Graph Extraction steps via Temporal workflow status polling.

### Compliance & Audit View
- A matrix view generated by the Compliance Agent.
- Rows: Regulations (e.g., OSHA 1910). Columns: Plant Areas. Cells: Red/Green status based on latest inspection cross-referencing.
